# レビュー品質分析ツール (Review Quality Analyzer)

ソフトウェア開発のレビュー指摘データ（Excel）を読み込み、品質傾向の可視化とAIによる分析・改善策提案を行うReactアーティファクトです。

Claude.ai上でアーティファクトとして動作します。

## 概要

設計ドキュメントのレビューで記録された指摘データを多角的に分析し、チームの品質課題を可視化します。Claude APIと連携したAI分析により、データに基づく具体的な改善策の提案や、担当者ごとの強み・課題の評価も可能です。

## 使い方

1. Claude.aiでこのファイル（`review-quality-analyzer.jsx`）をアーティファクトとして表示します
2. アーティファクト上にExcelファイル（`.xlsx` / `.xls`）をドラッグ＆ドロップ、またはクリックして選択します
3. 自動的に列名を認識し、ダッシュボードが表示されます
4. タブを切り替えて各種分析を確認できます
5. 「AI分析・改善策」タブや「担当者別」タブのAI評価ボタンで、Claude APIによる分析を実行できます

下記URL上で実行可能です
https://claude.ai/public/artifacts/779bac73-a9dc-4251-b3f6-2b5c77fd7172

## 主な機能

### ダッシュボード（8タブ構成）

| タブ | 内容 |
|------|------|
| 概要 | 総指摘件数、最多分類・原因、ドキュメント数のサマリーカード＋円グラフ |
| 指摘一覧 | 全指摘のカード表示。キーワード検索・分類/原因/文書/担当者/工程のフィルタリング・ページネーション対応 |
| 不具合分類 | 分類別の横棒グラフ＋パレート表（件数・割合・累積割合） |
| 原因分析 | 原因分布の棒グラフ＋レーダーチャート |
| 混入工程 | 工程別の円グラフ・棒グラフ＋工程×分類/原因のクロス集計 |
| ドキュメント別 | 文書別指摘件数＋文書×分類のクロス集計 |
| 担当者別 | 担当者別件数・クロス集計＋AI担当者評価（強み・課題・改善アクション） |
| AI分析・改善策 | Claude APIによる全体傾向分析と具体的な改善策提案 |

### AI連携機能

- **全体分析**: 不具合分類・原因・ドキュメント別の傾向から改善策を提案
- **担当者評価**: 担当者ごとの指摘傾向を分析し、強み・課題・改善アクションをカード形式で表示（リスクレベルのバッジ付き）
- **カスタムプロンプト**: 追加の分析観点や重点ポイントをユーザーが自由に指定可能

## Excelファイルの形式

### 必須列

| 列 | 例 |
|----|-----|
| 不具合分類 | 内容漏れ, 内容誤り, 規約違反 等 |
| 不具合原因 | 業務習熟不足, 不注意, 検討不十分 等 |
| 指摘内容 | 自由記述テキスト |
| ドキュメント名 | 認証機能 詳細設計書 等 |

### 任意列

| 列 | 用途 |
|----|------|
| 担当者 | 担当者別タブ・AI担当者評価 |
| 混入工程 | 混入工程タブ |

## サンプルファイル

動作確認用のサンプルExcelファイルを3つ用意しています（`doc/sample/`）。

| ファイル | 内容 | 件数 |
|----------|------|------|
| `レビュー指摘一覧_サンプル_担当者集中パターン.xlsx` | 特定の担当者（高橋翔太）に指摘が集中したデータ | 43件 |
| `レビュー指摘一覧_サンプル_内容漏れ集中パターン.xlsx` | 不具合分類「内容漏れ」が約65%を占めるデータ | 43件 |
| `レビュー指摘一覧_サンプル_100件.xlsx` | 汎用的なサンプルデータ | 100件 |

いずれも混入工程は全て「詳細設計」です。

また、`doc/template/` にExcel入力用のテンプレートファイル（`レビュー指摘一覧テンプレート.xlsx`）も用意しています。

## 技術スタック

| 技術 | 用途 |
|------|------|
| React | UIフレームワーク |
| Recharts | グラフ描画（円グラフ、棒グラフ、レーダーチャート） |

## 制限事項

- Claude.aiのアーティファクト環境で動作します
- AI分析機能を使用するにはClaude.aiにログインしている必要があります。未ログイン状態では集計・可視化のみ利用可能です
- 指摘内容のAI送信はサンプル50件（各80文字）に制限されています
- 対応ファイル形式は `.xlsx` および `.xls` のみです
